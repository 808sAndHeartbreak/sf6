使用说明

📌 这是什么？

SF6 FRAME LAB 是一款卡帧套路压制计算工具，帮助找到最优的卡帧压制组合方案。

🎮 如何使用

  1. 选择角色 - 点击右上角角色按钮切换
  2. 查看卡帧策略 - SOLUTION 页签展示各种击倒情境下的最优方案
  3. 添加场景 - 点击 + 添加场景 输入有利帧数，系统自动计算
  4. 管理帧数据 - DATA 页签可查看/导入角色招式数据

💡 方案解读

  • 卡帧招式 → 用来消耗多余的有利帧
  • 压制招式 → 对手起身时攻击命中的招式
  • H +X → 命中时的帧数优势
  • B +X/-X → 防御时的帧数优势
  • 偷 XF → 比正常更早命中，多偷的帧数

⭐ 收藏功能

点击方案旁的 ☆ 可收藏个人喜欢的方案，收藏后会优先显示并高亮。

🎯 期望后续 (进阶)

添加场景时可展开"期望后续"选项，支持两种模式：

  • • 指定剩余帧数 - 输入期望卡帧后剩余的帧数，验证是否有可行方案
  • • 指定压制招式 - 选择想用的压制招式，系统计算如何卡帧（需偷至少1帧）
  • • 若无可行方案，将提示并阻止添加

🚫 组合规则

在 DATA 页签底部可以管理组合过滤规则：

  • • GLOBAL 规则对所有角色生效（如轻拳+轻拳禁止）
  • • 角色专属 规则只对当前角色生效
  • • 点击 + 添加规则 可添加自定义规则

⌨️ 快捷键

  • • Esc 关闭弹窗
  • • Enter 确认保存
