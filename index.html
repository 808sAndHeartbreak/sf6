使用说明

📌 这是什么？

SF6 FRAME LAB 是一款卡帧套路压制计算工具，帮助找到最优的卡帧压制组合方案。

🎮 如何使用

  1. 选择角色 - 点击右上角角色按钮切换
  2. 查看卡帧策略 - SOLUTION 页签展示各种击倒情境下的最优方案
  3. 添加场景 - 点击 Add Scenario 输入有利帧数，系统自动计算
  4. 管理帧数据 - FRAME DATA 页签可查看/导入角色招式数据

💡 方案解读

  • 卡帧招式 → 用来消耗多余的有利帧
  • 压制招式 → 对手起身时攻击命中的招式
  • H +X → 命中时的帧数优势
  • B +X/-X → 防御时的帧数优势
  • 偷 XF → 比正常更早命中，多偷的帧数

⭐ 收藏功能

点击方案旁的 ☆ 可收藏个人喜欢的方案，收藏后会优先显示并高亮。
